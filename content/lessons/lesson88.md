# AIでクイズを作ろう

AIにクイズを作ってもらいましょう。

まず、Geminiを開きましょう。

https://gemini.google.com/app?hl=ja

たとえば次のようなお願いをしてみましょう。

```
きのこたけのこ戦争のクイズを5問作って。
```
```
小学校3年生のために漢字のクイズを作って。
```
```
マインクラフトのマニアのためにむずしい問題を10問作って。
```

次のようにおねがいすれば、答えをえらぶのではなく、じぶんで入れるようにすることもできます。

あなたはクイズ番組の司会者です。
私を挑戦者だと思って、1問ずつ順番に「都道府県クイズ」を出してください。
全部で5問出してください。
私が答えたら、正解かどうかを判定して、問題の解説をして、次の問題を出してください。
