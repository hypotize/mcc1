# AIで物語を作ろう

AIに物語を作ってもらいましょう。

まず、Geminiを開きましょう。

https://gemini.google.com/app?hl=ja

たとえば次のようなお願いをしてみましょう。

```
あなたは物語を作るプロです。
次の3つのものを使って、短い物語を書いてください。
小学生が読めるように、やさしい言葉を使ってください。

主人公：空飛ぶペンギン

舞台：雲の上にある学校

できごと：ある日空を飛べなくなった
```

主人公、舞台、できごとを自由に変えてAIにお願いしてみましょう。

# AIにアイデアを出してもらいましょう

AIに物語のアイデアを出してもらうこともできます。

```
小学生向けの物語を作りたい。主人公、舞台、できごとについて、アイデアを3つ出してください。
```

いいアイデアがあったら、それを使って物語を作ってみましょう。

# 物語をおもしろくするアイデアを出してもらいましょう

```
この物語をおもしろくするアイデアを3つ出してください。
```

もし、2番めのアイデアがよければ、次のようにお願いしてみましょう。

```
2番めのアイデアを使って物語を変えてください。
```
