# いろんなサイトで遊んでみよう

## ゲーム
- Ri-li

[https://ri-li.sourceforge.net/](https://ri-li.sourceforge.net/)

電車を自分自身に衝突させないように走らせるゲーム。
スネークゲームに似ている。
操作は画面に表示される矢印を、電車を走らせたい方向に矢印キーで変えるだけ。

`sudo apt install ri-li`

- Draw a Perfect Circle

[https://neal.fun/perfect-circle/](https://neal.fun/perfect-circle/)

中心点に合わせて円を正確に描いて正確さを競う単純なゲーム。
みんなで競争しよう。

- 数字記憶テスト

[https://www.arealme.com/number-memory-test/ja/](https://www.arealme.com/number-memory-test/ja/)

画面に表示される数字の位置を1から順番に憶えてクリックするゲーム。
結構難しい。

- スクラッチでスイカゲームを作る動画

[https://www.youtube.com/watch?v=IpT0-zUGtQw](https://www.youtube.com/watch?v=IpT0-zUGtQw)

それほど難しくなくて、見た目はおもしろい。

- newdoku

[https://newdoku.com/include/online.php?lang=ja](https://newdoku.com/include/online.php?lang=ja)

数独の変形。難易度をいろいろ変えられる。

- 正規表現パズル

[https://gametsukurukun.com/regexp-puzzle/]

パズルを解くように正規表現を勉強できる。結構難しい。

## プログラミング

- 異世界ニートエンジニア英雄になる

[https://ppg.paiza.jp/isekai?type=ppg_text](https://ppg.paiza.jp/isekai?type=ppg_text)

今作は主人公を育成し、魔王を討伐する、シリーズ初の育成ゲームです。
訓練（問題への解答）を通して、主人公の「アルゴリズム」「データ構造」「処理速度」「コード量」「SQL」能力を育成します。

- Pythonの教材

[筑波大学](https://mitani.cs.tsukuba.ac.jp/book_support/python/)

[東京大学](https://utokyo-ipp.github.io/)

[京都大学](https://repository.kulib.kyoto-u.ac.jp/dspace/bitstream/2433/265459/1/Version2021_10_08_01.pdf)

説明が詳しい　演習問題が多い

- Automate the Boring Stuff with Python

[「退屈なことはpythonにやらせよう」の原著](https://automatetheboringstuff.com/)

基礎から初めて、画像編集、スクレイピング、メール送信などを扱っている。

- Python,Pygameでゲームを作る

[https://inventwithpython.com/invent4thed/](https://inventwithpython.com/invent4thed/)

[https://inventwithpython.com/pygame/](https://inventwithpython.com/pygame/)

「退屈なことはpythonにやらせよう」と同じ作者の本

- paizaラーニング

[https://paiza.jp/works](https://paiza.jp/works)

- TOYPRO　小学生から始めるPython競技プログラミング

[https://www.toy-pro.net/](https://www.toy-pro.net/)

自習には向いている。

- プロジェクトオイラー

[https://projecteuler.net/](https://projecteuler.net/)

[日本語翻訳ページ](http://www.odz.sakura.ne.jp/projecteuler/)

数学の問題をプログラミングで解く
小学生には少し難しい。
言語は限定していない。
正解を出さないと解説が読めない

[プログラミング体験ゲーム：アルゴロジック　｜　JEITA](https://algo.jeita.or.jp/)
